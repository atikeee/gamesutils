✏️ Edit Bookmarks

Format Guide

Structure
Topic Subtopic Link name,https://... Link name,https://...
Indent Rules
0 spaces → Topic
4 spaces → Subtopic
8 spaces → Link, URL
Use comma or tab to separate name from URL.
Comments & Blanks
Lines starting with # are ignored.
Blank lines are ignored too.
Example
Learning Python docs,https://docs.python.org tutorial,https://realpython.com Linux man pages,https://man7.org
Tips
• Save with Ctrl+S
• Tab key inserts 4 spaces
• Changes are live on /links immediately after save
